# 합성 건강검진 데이터 생성기 (synthetic)

KOSIS 국민건강보험공단 건강검진통계(2024)의 **집계 분포**로부터 개인 단위 가상
검진 레코드를 생성하는 독립 서브프로젝트. 외부 라이브러리 없이 **파이썬 표준
라이브러리만** 사용한다. 실존 인물과 무관한 합성 데이터이며 연구·개발·교육용이다.

## 빠른 시작

```powershell
# 대시보드 (http://localhost:8081)
python synthetic/server.py

# CLI 생성 (CSV 3종 저장)
python synthetic/generator.py --n 10000 --sido 서울특별시 --seed 42 --out-dir output

# 시군구 단위 (코드 또는 이름)
python synthetic/generator.py --n 5000 --sigungu 11110 --out-dir output

# 테스트
python synthetic/test_generator.py
```

표본 수 상한은 **500,000행**(50만 행은 생성+검증에 수 분 소요).
배포본: https://hc-mkdata.tms-ai-lab.com (구글 로그인, 승인 이메일만)

최초 실행 시 `data/synthetic_baseline.json` 스펙 캐시를 자동 빌드한다(약 1초).
KOSIS CSV가 갱신되면 `--rebuild-spec`으로 재빌드.

## 산출물

| 안 | 내용 | 다운로드 |
|---|---|---|
| A안 | 개인별 32컬럼 합성 레코드 (인구학+체형+혈압/혈당/지질/간/신장/기타+판정, 시군구 모드 시 `sigungu` 채움) | `/api/download?type=a` |
| B안 | 검사항목 12종 × 정상/주의/질환의심 비율 요약 | `/api/download?type=b` |
| C안 | 시도×연령대 위험군 매트릭스 (비만·고혈압·당뇨·이상지질 + 종합위험) | `/api/download?type=c` |

연령은 **1세 단위(`age`)와 5세 밴드(`age_group`), 10세 연령대(`age_decade`)**를
동시 제공한다.

## 방법론

1. **인구학 결합분포**: 시군구 인구피라미드(`pop_pyramid.json`)를 시도로 합산해
   시드로 쓰고, KOSIS 수검인원 마진(연령밴드×성별 `N002_1`, 시도×성별 `N056`)에
   **IPF**(반복비례적합)로 맞춘 (시도×연령밴드×성별) 분포에서 추출.
   나이는 밴드 내 피라미드 가중으로 1세 단위 부여.
2. **검사수치 — 가우시안 코퓰라 + 분위수 매핑**: 항목별 잠재 정규변수에 의료적
   상관 네트워크(`LATENT_DEFS`: BMI→허리둘레/혈압/혈당/지질/간수치, 좌안→우안 시력,
   위험도→유질환자)를 부여. 교차공분산을 정확히 전파해 각 잠재변수가 단위분산을
   유지하므로, (연령밴드×성별) KOSIS 구간분포를 (시도×성별) 분포로 **레이킹**한
   목표분포의 역CDF에 분위수 매핑하면 **주변분포가 KOSIS와 일치**한다.
3. **임상 파생값 (개인 내 일관성)**: 실제 검진 결과지에서 결정함수로 산출되는 두
   항목은 코퓰라가 아니라 식으로 직접 파생해 개인 단위 정합성을 보장한다.
   - **LDL** = `TC − HDL − TG/5` (Friedewald, 한국 검진 표준; TG≥400은 식이 부정확해
     KOSIS 분포에서 분위수 매핑). → (TC,HDL,TG,LDL) 4중 일관성 완전 보장.
   - **eGFR** = CKD-EPI 2021(크레아티닌·연령·성별, 인종 비포함). → (Cr,eGFR) 일관성 보장.

     이 두 값은 식으로 결정되므로 KOSIS 주변분포(N076/77, N128/129)와는 다소 어긋날
     수 있다(검증 탭에 "파생값"으로 표시).
4. **판정**: `result_grade`(정상A/정상B/질환의심)는 검진 기준 규칙으로 산출하고,
   **유질환자**는 판정현황(`N099`)의 셀별 비율을 위험 잠재변수와 결합한 임계모형으로
   부여(주변비율 보존 + 고위험자일수록 유질환자일 확률↑).
   - 정상A: 건강 양호 / 정상B: 이상 없으나 자기관리·예방조치 필요(경계) /
     질환의심: 2차 정밀검사 필요 / 유질환자: 현재 치료 중인 질환 보유.
5. **시군구 모드**: 인구학(연령·성·1세 나이)은 「시군구 인구피라미드 × 소속 시도
   수검률」 보정 분포에서 추출하고, **검사수치 분포는 소속 시도 기준**을 쓴다
   (KOSIS가 검사수치를 시도·연령까지만 제공). A안에 `sigungu` 컬럼이 채워진다.

## 정합성 (이중 기준)

대시보드 [검증] 탭은 두 기준으로 KOSIS 대비 충실도를 보여준다.

- **내부 정합성**(분위수 매핑 충실도, 코호트 구성 통제): 비파생 항목 평균 ≈0.16%p,
  구간 최대 ≤1%p. 사실상 샘플링 잡음 수준.
- **KOSIS 원시 전국 대비**(발행 분포 직접 비교, 코호트 연령·시도 구성 차이 포함):
  평균 ≈0.8%p. 이 값을 KPI 헤드라인으로 표시한다.
- **파생 항목**(LDL·eGFR): 임상 공식으로 산출하므로 KOSIS 주변분포와 의도적으로
  다를 수 있다(LDL ~1.8%p, eGFR ~10%p). 검증 탭에 "파생값"으로 별도 표기.

## 가정과 한계

- **신장 SD**: KOSIS는 평균만 제공 → 연령대·성별 SD(4.8~6.0cm)를 가정.
  체중은 `BMI×(신장/100)²`로 유도(개인 내 일관성 보장). 체중 평균(N132)은 검증용으로만.
- **LDL·eGFR**: 임상 결정식으로 파생(위 방법론 3)하므로 KOSIS 주변분포와 어긋날 수
  있으나 개인 내 일관성은 완전. 주변분포 충실도보다 임상 정합성을 우선한 설계 선택.
- **밴드 내 나이**: 5세 피라미드 밴드를 균등 분할(1세 단위 정밀도는 근사).
- **빈 셀 폴백**: 지질 등 검사주기 제한으로 비는 연령셀(예: 19세 이하 LDL)은
  가장 가까운 밴드 분포로 폴백, 빈 시도셀은 전국 분포로 폴백.
- **시도×연령 상호작용**: 분포 레이킹은 시도 효과를 곱셈 보정으로만 반영
  (시도×연령×성별 3원 분포는 KOSIS에 없음).
- **골밀도**: 실제 검진 프로그램과 동일하게 54·66세 여성만 값 보유(시도별 분포 N094).
- **판정등급**: 정상A/B·질환의심은 규칙 기반이라 KOSIS 판정현황과 수 %p 차이가
  있을 수 있다(유질환자만 N099로 보정). 대시보드 검증 탭에 참고용으로 병기.
- **시군구 모드**: 검사수치는 시도 분포의 근사 — 시군구 간 검사수치 차이(예: 도시
  vs 농촌 비만율 격차)는 반영되지 않고 연령·성 구조 차이를 통한 간접 효과만 반영.
  또한 **시도 수검률을 시군구에 균일 적용**하므로 시군구별 수검률(uptake) 차이
  (예: 의료접근성이 낮은 군 지역의 낮은 수검률)도 반영되지 않는다.
- **지역(시도) 간 차이**: C안에서 보이는 시도 간 격차는 KOSIS 시도별 분포에 실재하는
  격차(예: 제주·전남 비만율 상위, 세종 하위)를 충실히 따른 것이다. 신뢰도 탭의
  「시도별 비만율 — 지역 격차 검증」 카드에서 합성 vs KOSIS를 직접 비교할 수 있다.
- 같은 `seed` → 완전 재현. 항목 간 상관강도는 `corr`(0~1.5)로 조절(0=독립).

## 재식별 위험성평가 지표

- **k-익명성**: 판정 기준 **k≥3** — 연구·내부 공유에서 통용되는 실무 하한(구 비식별
  조치 가이드라인(2016) 적정성평가 사례 수준). 현행 가명정보 처리 가이드라인은 고정
  최소 k를 규정하지 않으며, 보건의료 데이터엔 k≥5가 보수적 통념이라 **k≥5 충족률을
  병기**한다. 준식별자는 지역(시도 또는 시군구)+연령(1세/5세/10세)+성별 3종.
- **l-다양성·t-근접성**: 민감속성을 종합판정(`result_grade`)으로 두고 측정.
  t는 순서형 EMD(0~1)이며, 연령↔판정의 실제 상관 때문에 0이 되지 않는 것이 정상
  (KOSIS 공표 통계에 이미 있는 집단 수준 정보로, 개인 누출이 아님).
